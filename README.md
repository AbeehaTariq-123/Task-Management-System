# Task Management System

## Features

- Full CRUD Operations
- JWT Authentication
- MongoDB Database
- Search Tasks
- Task Progress Tracking
- Responsive UI
- REST API
- React Frontend
- Express Backend

---

# Backend Setup

```bash
cd backend
npm install
npm run dev

# Frontend Setup

cd frontend
npm install
npm run dev
